# YourWork - 企业项目管理系统开发设计方案

## 一、项目概述

### 1.1 项目名称
YourWork - 企业项目化管理及日常办公软件工具集合

### 1.2 项目目标
构建一个集成化的企业管理平台，包含：
- 项目化管理工具（一期核心）
- 日程表工具（二期）
- OA办公工具（三期）

### 1.3 设计原则
- **最小依赖**：只使用必要的技术，能原生就原生
- **命名统一**：全程 snake_case，无转换
- **调试友好**：所有代码可追踪，日志清晰
- **简单直接**：避免过度工程化

## 三、为什么选择这些技术

### 3.1 为什么必须用 FastAPI？

**问题**：纯 Python 能处理 HTTP 请求吗？

**答案**：不能。Python 本身只是一个编程语言，需要 Web 框架来：
- 监听端口（如 8000）
- 接收 HTTP 请求
- 解析请求参数
- 返回 HTTP 响应

**选择 FastAPI 的理由**：
1. 代码量少，语法简单
2. 自动生成 API 文档（访问 /docs）
3. 性能优秀
4. 异步支持

**FastAPI 核心代码示例**：
```python
# main.py - 这是全部代码，非常简单
from fastapi import FastAPI
from fastapi.responses import HTMLResponse, FileResponse
from fastapi.staticfiles import StaticFiles
import sqlite3
import json

app = FastAPI()

# 静态文件
app.mount("/static", StaticFiles(directory="static"), name="static")

# 数据库连接 - 使用原生 sqlite3
def get_db():
    conn = sqlite3.connect('yourwork.db')
    conn.row_factory = sqlite3.Row  # 返回字典格式
    return conn

# 页面路由 - 返回HTML
@app.get("/", response_class=HTMLResponse)
async def index():
    with open("templates/index.html", "r", encoding="utf-8") as f:
        return f.read()

# API路由 - 返回JSON
@app.get("/api/v1/projects")
async def get_projects():
    conn = get_db()
    cursor = conn.execute("SELECT * FROM projects")
    projects = [dict(row) for row in cursor.fetchall()]
    conn.close()
    return {"code": 0, "data": projects}

@app.post("/api/v1/projects")
async def create_project(project: dict):
    conn = get_db()
    conn.execute(
        "INSERT INTO projects (name, status) VALUES (?, ?)",
        (project['name'], 'in_progress')
    )
    conn.commit()
    conn.close()
    return {"code": 0, "message": "创建成功"}

# 启动命令: uvicorn main:app --reload
```

### 3.2 为什么用原生 sqlite3 而不是 ORM？

**原生 sqlite3 优点**：
- Python 内置，无需安装
- SQL 语句清晰可见，便于调试
- 没有额外的抽象层

**示例对比**：
```python
# ORM 方式（需要学习 SQLAlchemy）
project = session.query(Project).filter(Project.id == project_id).first()

# 原生 SQL（清晰直接）
cursor = conn.execute("SELECT * FROM projects WHERE id = ?", (project_id,))
project = cursor.fetchone()
```

### 3.3 为什么不用 Bootstrap？

**Bootstrap 问题**：
- 引入 200KB+ 的 CSS 文件
- 样式不可控，难以调试
- 依赖外部 CDN 或本地文件

- 完全可控
- 代码量小
- 调试简单

```python
logging.basicConfig(

    datefmt='%H:%M:%S',

logger = logging.getLogger(__name__)

# ===== 数据库配置（原生 sqlite3）=====
DB_PATH = "yourwork.db"

def get_db():
    """获取数据库连接"""
    conn = sqlite3.connect(DB_PATH)
    conn.row_factory = sqlite3.Row  # 返回字典格式
    return conn

def init_db():
    """初始化数据库表"""
    conn = get_db()
    conn.executescript("""
        -- 用户表
        CREATE TABLE IF NOT EXISTS users (
            id TEXT PRIMARY KEY,
            username TEXT UNIQUE NOT NULL,
            password TEXT NOT NULL,
            display_name TEXT,
            email TEXT,
            avatar TEXT,
            is_active INTEGER DEFAULT 1,
            created_at TEXT,
            updated_at TEXT
        );

        -- 角色表
        CREATE TABLE IF NOT EXISTS roles (
            id TEXT PRIMARY KEY,
            name TEXT NOT NULL,
            code TEXT UNIQUE NOT NULL,
            description TEXT,
            is_system INTEGER DEFAULT 0
        );

        -- 用户角色关联表
        CREATE TABLE IF NOT EXISTS user_roles (
            id TEXT PRIMARY KEY,
            user_id TEXT NOT NULL,
            role_id TEXT NOT NULL,
            FOREIGN KEY (user_id) REFERENCES users(id),
            FOREIGN KEY (role_id) REFERENCES roles(id)
        );

        -- 项目表
        CREATE TABLE IF NOT EXISTS projects (
            id TEXT PRIMARY KEY,
            project_no TEXT UNIQUE NOT NULL,
            name TEXT NOT NULL,
            status TEXT DEFAULT 'in_progress',
            resources TEXT,
            created_at TEXT,
            updated_at TEXT
        );

        -- 项目成员表
        CREATE TABLE IF NOT EXISTS project_members (
            id TEXT PRIMARY KEY,
            project_id TEXT NOT NULL,
            user_id TEXT NOT NULL,
            roles TEXT,
            display_name TEXT,
            FOREIGN KEY (project_id) REFERENCES projects(id),
            FOREIGN KEY (user_id) REFERENCES users(id)
        );

        -- 里程碑表
        CREATE TABLE IF NOT EXISTS milestones (
            id TEXT PRIMARY KEY,
            project_id TEXT NOT NULL,

            parent_id TEXT,
            execution_result TEXT,
            created_at TEXT,
            updated_at TEXT,

        -- 产出物表
        CREATE TABLE IF NOT EXISTS deliverables (
            id TEXT PRIMARY KEY,
            name TEXT NOT NULL,
            original_name TEXT NOT NULL,
            file_path TEXT NOT NULL,
            file_size INTEGER,
            file_type TEXT,

            created_at TEXT,

            type TEXT NOT NULL,

            created_at TEXT,
            FOREIGN KEY (user_id) REFERENCES users(id)

# ===== 工具函数 =====
def generate_id():
    """生成唯一ID"""
    return str(uuid.uuid4())

def hash_password(password: str) -> str:
    """密码加密（原生 hashlib）"""
    return hashlib.sha256(password.encode()).hexdigest()

def verify_password(password: str, hashed: str) -> bool:
    """验证密码"""
    return hash_password(password) == hashed

def get_current_user(request: Request) -> Optional[dict]:
    """获取当前登录用户"""
    token = request.cookies.get('token')
    if not token:
        return None
    # 简单实现：token 就是 user_id
    conn = get_db()
    cursor = conn.execute("SELECT * FROM users WHERE id = ?", (token,))
    user = cursor.fetchone()
    conn.close()
    return dict(user) if user else None

@app.post("/api/v1/auth/login")
async def login(request: Request, response: Response):
    """用户登录"""
    data = await request.json()
    username = data.get('username')
    password = data.get('password')

    logger.info(f"登录尝试: {username}")

    conn = get_db()
    cursor = conn.execute(
        "SELECT * FROM users WHERE username = ?",
        (username,)
    )
    user = cursor.fetchone()
    conn.close()

    if not user or not verify_password(password, user['password']):
        logger.warning(f"登录失败: {username}")
        return {"code": 401, "message": "用户名或密码错误"}

    # 设置 Cookie（简单实现，不用 JWT）
    response.set_cookie(
        key="token",
        value=user['id'],
        httponly=True,
        max_age=7 * 24 * 60 * 60  # 7天
    )

    logger.info(f"登录成功: {username}")
    return {
        "code": 0,
        "message": "登录成功",
        "data": {
            "user_id": user['id'],
            "username": user['username'],
            "display_name": user['display_name']
        }
    }

@app.post("/api/v1/auth/logout")

@app.get("/api/v1/projects")
async def get_projects(request: Request):

    if not user:
        return {"code": 401, "message": "未登录"}

    conn = get_db()

    projects = [dict(row) for row in cursor.fetchall()]
    conn.close()

    logger.info(f"返回 {len(projects)} 个项目")
    return {"code": 0, "data": projects}

@app.post("/api/v1/projects")
async def create_project(request: Request):
    """创建项目"""
    user = get_current_user(request)
    if not user:
        return {"code": 401, "message": "未登录"}

    data = await request.json()
    project_name = data.get('name')

    logger.info(f"创建项目: {project_name}, user={user['username']}")

    project_id = generate_id()
    project_no = f"PRJ-{datetime.now().strftime('%Y%m%d')}-{project_id[:8]}"
    now = datetime.now().isoformat()

    conn = get_db()
    conn.execute(
        "INSERT INTO projects (id, project_no, name, status, created_at, updated_at) VALUES (?, ?, ?, ?, ?, ?)",
        (project_id, project_no, project_name, 'in_progress', now, now)
    )
    conn.commit()
    conn.close()

    logger.info(f"项目创建成功: {project_no}")
    return {
        "code": 0,
        "message": "创建成功",
        "data": {
            "project_id": project_id,
            "project_no": project_no
        }
    }

@app.get("/api/v1/projects/{project_id}")
async def get_project(project_id: str, request: Request):
    """获取项目详情"""
    user = get_current_user(request)
    if not user:
        return {"code": 401, "message": "未登录"}

    logger.info(f"获取项目详情: {project_id}")

    conn = get_db()

    project = cursor.fetchone()

    # 获取里程碑

        (project_id,)
    )
    milestones = [dict(row) for row in cursor.fetchall()]

    # 获取成员
    cursor = conn.execute(
        "SELECT * FROM project_members WHERE project_id = ?",
        (project_id,)
    )
    members = [dict(row) for row in cursor.fetchall()]

    conn.close()

    return {
        "code": 0,
        "data": {
            "project": dict(project),
            "milestones": milestones,
            "members": members
        }
    }

# ===== 启动 =====
if __name__ == "__main__":
    import uvicorn

    # 确保目录存在
    os.makedirs("logs", exist_ok=True)
    os.makedirs("uploads/projects", exist_ok=True)
    os.makedirs("static/css", exist_ok=True)
    os.makedirs("static/js", exist_ok=True)
    os.makedirs("templates", exist_ok=True)

    # 初始化数据库
    if not os.path.exists(DB_PATH):
        init_db()
        logger.info("数据库文件创建完成")

    logger.info("=" * 50)
    logger.info("YourWork 服务启动")
    logger.info("访问地址: http://localhost:8000")
    logger.info("API文档: http://localhost:8000/docs")
    logger.info("=" * 50)

    uvicorn.run(app, host="0.0.0.0", port=8000)
```

### 5.4 前端 JavaScript (static/js/api.js)

```javascript
/**
 * API 封装 - 使用原生 Fetch API（浏览器内置）
 */

// API 基础路径
const API_BASE = '/api/v1';

/**
 * 日志函数 - 与后端日志格式保持一致
 */
function log(level, module, message, data) {
    const time = new Date().toTimeString().split(' ')[0];
    const prefix = '[' + level + '] ' + time + ' | ' + module + ' | ';

    if (data) {
        console.log(prefix + message, data);
    } else {
        console.log(prefix + message);
    }
}

/**
 * GET 请求
 */
async function api_get(endpoint) {
    const url = API_BASE + endpoint;
    log('INFO', 'API', 'GET ' + url);

    const start_time = Date.now();

    try {
        const response = await fetch(url);
        const data = await response.json();
        const duration = Date.now() - start_time;

        log('INFO', 'API', '响应 ' + url + ' [' + response.status + '] (' + duration + 'ms)', data);

        return data;
    } catch (error) {
        log('ERROR', 'API', '请求失败: ' + url, error);
        throw error;
    }
}

/**
 * POST 请求
 */
async function api_post(endpoint, body) {
    const url = API_BASE + endpoint;
    log('INFO', 'API', 'POST ' + url, body);

    const start_time = Date.now();

    try {
        const response = await fetch(url, {
            method: 'POST',
            headers: {
                'Content-Type': 'application/json'
            },
            body: JSON.stringify(body)
        });
        const data = await response.json();
        const duration = Date.now() - start_time;

        log('INFO', 'API', '响应 ' + url + ' [' + response.status + '] (' + duration + 'ms)', data);

        return data;
    } catch (error) {
        log('ERROR', 'API', '请求失败: ' + url, error);
        throw error;
    }
}

/**
 * 文件上传
 */
async function api_upload(endpoint, form_data) {
    const url = API_BASE + endpoint;
    log('INFO', 'API', 'UPLOAD ' + url);

    const start_time = Date.now();

    try {
        const response = await fetch(url, {
            method: 'POST',
            body: form_data
        });
        const data = await response.json();
        const duration = Date.now() - start_time;

        log('INFO', 'API', '响应 ' + url + ' [' + response.status + '] (' + duration + 'ms)', data);

        return data;
    } catch (error) {
        log('ERROR', 'API', '上传失败: ' + url, error);
        throw error;
    }
}

/**
 * 检查登录状态
 */
async function check_login() {
    const result = await api_get('/api/v1/auth/profile');
    if (result.code === 0) {
        return result.data;
    }
    return null;
}
```

## 十、总结

### 核心原则
1. **最小依赖**：只用 FastAPI，其他全部原生
2. **代码可见**：所有代码直接可读，无抽象层
3. **调试简单**：命令行 + 浏览器控制台即可

### 技术选择理由
- **FastAPI**：必须，处理 HTTP 请求
- **sqlite3**：Python 内置，无需安装
- **hashlib**：Python 内置，密码加密
- **Fetch API**：浏览器内置，HTTP 请求
- **原生 CSS**：无依赖，完全可控
